Student Dashboard

Student Dashboard was designed as part of Credit Suisse Coding Challenge. We were ranked 6th in the competition.
Student Dashboard categorised users into 3 types : Teacher, Student and Admin Student Dashboard is a portal with the following features :
Students
1. Students have access to forum where they can discuss with each other on various academic as well as non academic topics.
2. Students can get important college announcments and news
3. Students can submit their assignment files via the portal
Teacher
1. Teacher can upload imporatant questions as well as submit new notices
I worked as Full Stack developer to develop a Student Dashboard. I designed the UI for the dashboard as well as handled the back end working of the application
