DoubleMap - Usability Evaluation

Double Map is an application designed to track shuttle services within the campus. Campus can vary from universities, corporate companies, hospitals and airports. We conducted usability evaluation of DoubleMap with the live version of web and mobile application. Android base smartphone and iPhone were used to test the android and ios version of the application and Google chrome for testing web based app. For usability evaluation we followed various methods like cognitive walkthrough, contextual inquiry, interviews, heuristic evaluation to collect data from for the evaluation


Research and Analysis

Competetive Analysis

Transloc Rider: It is a similar app that provides real time bus updates and announcements. Many transit-systems and university campuses use it

RideCell: It is a company that provides ride / car sharing applications. It also provides services for fixed routes fleet automation. Works like Transloc and DoubleMap.


Overall DoubleMap stands out with most features on paper as compared to competition. Thorough usability tests and evaluations we will analyze how these features perform.
