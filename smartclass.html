SmartClass


Design

Smart class is an android application which was designed to run on low cost android devices. It provides student and teachers a medium to interact with each other. Smart class has facilites like
1. Whiteboard : Teacher can share diagrams with students
2. File sharing : Teachers can send assignments or presentation files from the app
3. Chat room : Teacher can interact with all the students at once, a public chat room
4. Personal message : Student can talk with a particular teacher or teacher can talk with a particular student
The app was desinged under Aakash Software Development contest which was conducted by IIT Bombay. We were awarded the 2nd prize all over india for the application. I was the UI designer for android application in a 3 person team.
