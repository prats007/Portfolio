Research

Identify user behaviors and user goals using quantitative and qualitative methods. Discover and define user scenarios and mental models.

Design

Create a concept using Whiteboard ideation and brainstorming. Visualize the concepts using wireframes and prototypes.

Iterate

Validate designs through user testing. Modify designs after each round of testing until a winning solution is achieved
